# AI会社 — note自動化担当指示書

## 役割
note投稿の完全自動化を構築し、コンスタントなコンテンツ発信を実現する。

---

## 自動化する業務

### 1. 投稿スケジュール管理
- 週3本（月〜水 or 火〜木）の定期投稿
- 予約投稿システム構築
- 季節イベント・トレンドチェック

### 2. AI生成プロンプト管理

#### 記事生成用プロンプトテンプレート

```
【役割】あなたはキャリア教育×AI活用の専門家です。
【読者】大学3年生〜第二新股
【目的】{テーマ}について практичныеな解決法を提供し、信頼を構築する
【構成】
1. 导入（课题提起）: {悩み}の解决方案
2. 本文（知識提供）: ステップバイステップ
3. 具体例: 実際の案例
4. まとめ: 明日のアクション
5. CTA: メール登録/ASPリンク
【制約】
- 具体性と専門性重視
-  читатель視点の語りかけ口調
- SEO考虑のキーワード 配置
```

### 3. コンテンツパイプライン

```
[Career Expertからテーマを受信]
    ↓
[AIで記事初稿を生成]
    ↓
[Humanがレビュー・修正]
    ↓
[予約投稿実行]
    ↓
[パフォーマンス追跡]
```

## 品質チェック項目（投稿前確認）

- [ ] ターゲットに刺さるタイトルか
- [ ] 導入で本文を開きたくなるか
- [ ] 具体例・データが揃っているか
- [ ] CTAが明確か（ASPリンクの場所を確保）
- [ ] 文字数3000字以上
- [ ] キーワードseo対策済み
- [ ] 误字・脱字なし
