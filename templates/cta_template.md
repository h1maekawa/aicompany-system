# CTAテンプレート — 完成版

## 使用方法
記事タイプ（TOFU/MOFU/BOFU）に応じて、適切なCTAを選んで配置してください。
[] 内はプレースホルダーなので、実際のURLや文言に置き換えてください。

---

## TOFU記事用CTA（コールド読者向け）

### CTA-A：メール登録
```
【読者限定プレゼント】
忙しくて从何から始めればいいのかわからない...
そんなあなたのために、詳細な资料を作成しました。

メールアドレスを入力すると無料で受け取れます →
[MAIL_REGISTER_LINK]
```

### CTA-B：比較記事誘導
```
さらに詳しく比較したい場合は、
专门的比较記事を用意しています →
[COMPARE_ARTICLE_URL]
```

### CTA-C：note誘導
```
 notaで实演记事的を書いたので、
そちらも合わせて読んでみてください →
[NOTE_ARTICLE_URL]
```

**配置ルール：** TOFU記事のCTAは最大1回。本文末尾に配置。

## MOFU記事用CTA（ミドル読者向け）

### CTA-A：比較記事誘導
```
各サービスを比較した詳細な記事をご用意しました →
[COMPARE_ARTICLE_URL]
```

### CTA-B：メール登録（補助CTA）
```
サービスを選ぶ前に、まず资料で基本信息を押さえてませんか？
 →
[MAIL_REGISTER_LINK]
```

### CTA-C：A8リンク（比较後の自然流れ）
```
本気で比较して选びたいなら、このサービスがおすすめです →
[A8_LINK_XXX]
```

**配置ルール：** MOFU記事のCTAは最大2回。本文中1回 + 末尾1回。

## BOFU記事用CTA（ホット読者向け）

### CTA-A：メインCTA（本文中）
```
现在的读者層が求めていることを、最も有效地に解决できる服务是我们的推荐です →
[A8_LINK_XXX]
```

### CTA-B：比較後のCTA（まとめ前）
```
サービスを试すなら、まずは免费咨询から始めてみませんか？ →
[A8_LINK_XXX]
```

### CTA-C：最終CTA（まとめ直後）
```
犹豫しているなら、今が始め時です →
[A8_LINK_XXX]
```

**配置ルール：** BOFU記事のCTAは最大3回。本文中1回 + まとめ前1回 + まとめ直後1回。

## CTA 配置 检查表

- [ ] CTAが記事あたり3回以内
- [ ] TOFU記事のCTAはメール登録/比較誘導のみ
- [ ] MOFU記事の主CTAは比較誘導
- [ ] BOFU記事のCTAはA8リンクok
- [ ] CTA文言が夸大でない（「必ず」「絶対」なし）
- [ ] CTA 配置场所が自然（强行感なし）
- [ ] A8リンク先が正しいoffer_idと一致

## CTA 文言パターン集

### メール登録系
- 「详细资料を無料で受け取る」
- 「注册送られてくるレポート」
- 「条件を맞춤した方案を受け取る」

### 比較誘導系
- 「详细な比较表はこちら」
- 「各服务の评测結果一覧」
- 「最安値サービス一覧」

### A8リンク系
- 「今すぐ無料で咨詢」
- 「详细を見る・报名する」
- 「実際に试してみる」

### note誘導系
- 「实演记事的も公开しています」
- 「 notaでも詳しく解説」
- 「的其他記事はこちらから」
