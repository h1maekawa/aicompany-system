# note記事テンプレート — 完成版

## 使用方法
このテンプレートに従ってnote記事を生成してください。noteは「信頼構築」を目的とし、押し付けがましい销售色は避ける。

---

## タイトル
共感型・实演 型（「私がやったこと」ベース）
例：【実演】AI使ったSPI模擬テストで偏差値65達成した全過程

## 見出し構成

### 導入：悩みを打ち明ける

```
私は3ヶ月前、まさに[this problem]の状態でした。

[具体的な状態の描述]

- [ 问题1 ]
- [ 问题2 ]
- [ 问题3 ]

でも、ある方法を取り入れてから [具体的な结果] まで上げることができました。
```

### 本文：實際にやったこと

#### [段階1：刚开始]
本文：[ どのような状态下から始めたか ]
```
[hoge]
```

#### [段階2：方法论尝试]
本文：[ 尝试したこと、使ったツール ]
```
[ 具体的使用プロンプトや方法 ]
```

#### [段階3：结果が出る]
本文：[ 数字で示す変化 ]
```
- 開始時: [結果1]
- 1ヶ月後: [結果2]
- 3ヶ月後: [結果3]
```

### ビフォーアフター：数字で見せる

| 時期 | [指标] | 备注 |
|------|--------|------|
| 開始時 | ○○ | ... |
| 1ヶ月後 | ○○ | ... |
| 3ヶ月後 | ○○ | ... |

### 今日から始める方法

```
【手順1】
[ 明日の具体的なアクション ]

【手順2】
[ その次のアクション ]

【手順3】
[ 最終的な目標 ]
```

### まとめ：次のステップ

```
今日から、AIを[this topic]の武器にしましょう！
```

## CTA 配置ルール

### メール登録導線（1回）
本文末尾付近に自然に配置
```
【読者限定プレゼント】
「忙しくて从何から始めればいいのかわからない...」

そんなあなたのために、詳細な资料を作成しました。

メール地址を入力すると受け取れます →
[MAIL_REGISTER_LINK]
```

### 比較記事誘導（1回）
本文末尾に配置
```
さらに詳しく比較したい場合は、
专门的比较記事を用意しています →
[COMPARE_ARTICLE_URL]
```

### 注意：noteではA8リンク прямой 配置禁止
noteでは直接的なカウンセルは避け、あくまで「信頼構築」と「メール登録」を目的にする。
