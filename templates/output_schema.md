# 出力スキーマ — 全AI共通定義

## 使用方法
各AIが出力を生成する際、このスキーマに従って出力してください。
-markdown 表形式またはコードブロック形式で統一。

## スキーマ 使用ルール

1. 各AIは必ずこのスキーマに従って出力すること
2. 出力時は不要な項目は空欄に残さず、省略すること
3. 表形式またはコードブロック形式で統一すること
4. プレースホルダー（[XXX]）は実際の値に置き換えること
5. 品質チェック項目は出力末尾に必ず記載すること
