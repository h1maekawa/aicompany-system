## 2. スレッドテンプレート（5-10ポスト）

### ポスト1（フック）
```
[this topic]で悩んでいる人へ。

私は3ヶ月前に[this result]まで上げました。

具体的にやったことを全てお話します。
（以下スレッド）
```

### ポスト2〜9（内容）
```
【ポイント○】
[ 内容説明 ]
[ 具体例や数字 ]
```

### ポスト10（CTA + リンク）
```
全てを書くと长くなってしまうので、
詳細な方法は статья にまとめました →
[ARTICLE_URL]

何かあればコメントで聞いてください！
```

## 作成ルール

| 項目 | ルール |
|------|--------|
| 文字数 | 単発: 100-150字、スレッド: 各280字以内 |
| ハッシュタグ | 最大3つまで |
| 絵文字 | 2つ以内 |
| リンク | 必ず [ARTICLE_URL] プレースホルダを使用 |
| 禁止表現 | 「必勝」「絶対儲かる」「必ず届く」等 |
