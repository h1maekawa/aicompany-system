# SEO記事テンプレート — 完成版

## 使用方法
このテンプレートに従ってSEO記事を生成してください。[] 内はプレースホルダなので、実際のURLや文言に置き換えてください。

---

## タイトル
[ primary_keyword ] 包括、32文字以内
例：SPI対策 AI活用 完全ガイド【偏差値65達成者が解説】

## メタディスクリプション（170文字以内）
[ 記事を读完会有什么结果 ] が简潔に书かれた文章
例：SPI対策で偏差値65達成者が、AI使った効率的な勉强法を公开。プロンプト例付き。

---

## CTA 配置ルール

### 本文中（1回目）
自然に流れ落ちる箇所に配置
```
このような悩みをお持ちでしたら、[サービス名]の活用を検討してみてください。
→
[A8_LINK_XXX]
```

### まとめ前（2回目）
比較説明の後に配置
```
本気で変わるなら、今すぐ行動することです。
→
[A8_LINK_XXX]
```

### まとめ直後（3回目）
最終CTAとして配置
```
まずは無料资料をリクエストして雰囲気を確かめてみませんか？
→
[A8_LINK_XXX]
```

## 内部リンク 配置箇所

| 配置箇所 | 誘導先 | anchor_text |
|----------|--------|-------------|
| 方法説明の後 | TOFU記事 | [相關基礎知識] |
| 比較表の後 | MOFU比較記事 | [详细比較はこちら] |
| まとめ前 | BOFUレビュー | [当事人的声を見る] |
| まとめの中 | note記事 | [实演记事的を見る] |
| 末尾 | メール登録LP | [無料资料をリクエスト] |

## 品質チェックリスト

- [ ] タイトルに primary_keyword が含まれている
- [ ] メタディスクリプションが170文字以内
- [ ] 本文が5,000字以上
- [ ] H2/H3の见出し構成が明確
- [ ] 具体例・データが揃っている
- [ ] CTAが3回以内
- [ ] 内部リンクが5本以上
- [ ] A8リンク先が正しいoffer_idと一致
- [ ] 夸大表現（「必ず」「絶対」「最短」）がない
- [ ] 比較表が公平（片方不当上げなし）
