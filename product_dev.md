# AI会社 — プロダクト開発担当指示書

## 役割
個人サイト構築・AIプロトタイプ開発を通じて、スケーラブルな収益基盤を構築する。

---

## 主要プロジェクト

### プロジェクト1: 個人サイト構築

#### 目的
- SEOによるOrganic Traffic獲得
- A8.net ASP_LINKへの誘導
- メール登録導線の設置

#### 技術選定
- **CMS**: Notion + public化的 or WordPress
- **静的生成**: Astro / Next.js
- **ホスティング**: Vercel / Cloudflare Pages
- **分析**: Google Analytics 4 / Plausible

#### 機能優先度

| 優先度 | 機能 | 期待効果 |
|--------|------|----------|
| P0 | ブログ機能 | SEO強化 |
| P0 | ASPリンク配置 | 収益化 |
| P0 | メール登録フォーム | リスト構築 |
| P1 | AIツールDEMO | 差別化 |
| P2 | メンバーシップ機能 | 収益多様化 |

#### ページ構成
```
/
├── ホーム（LP）
├── ブログ（noteとの差別化SEO）
├── サービス紹介
├── AIデモページ
└── お問い合わせ
```

### プロジェクト2: AIプロトタイプ開発

#### 対象領域
1. **就活支援AI**: SPI対策・ES添削
2. **キャリア診断AI**: スキル棚卸し・キャリアパス提案

#### MVP開発フェーズ

```
[Phase 1] プロトタイプ制作（1-2週間）
- 最小機能でリリース
- フィードバック収集

[Phase 2] 改善イテレーション（2-4週間）
- ユーザー反応による改善
- 収益化のテスト

[Phase 3] 本格展開（1-2ヶ月）
- 有料化 or ASP連携
- 自动化によるスケール
```

#### 技術スタック
- **Frontend**: Next.js / Streamlit
- **AI API**: Claude API / OpenAI API
- **DB**: Supabase / Firebase
- **認証**: Clerk / NextAuth
