## SPIの本当の構造、知ってますか？

SPIには大きく分けて2つの分野があります。

### 言語領域（言語理解）
- 熟語の構成
- 文法・敬語
- 読解力
- 排列・長文読解

### 非言語領域（数的処理）
- 推論（順当・損益算）
- 場合の数・確率
- 図形の展開・集合
- 方程式・速度算

多くの就活生が陥るのは **「参考書の問題をすべて解こうとする」** 误区。

実際は、出題パターンを知り、頻出問題を反復練習するだけで十分です。

問題は、**自分に合った学習法が見つからない** こと。

そこで私がたどり着いたのが、**AI（ChatGPT）を活用した学習法** です。

## AIを活用したSPI学習法第

ここからは、実際に使ったプロンプト（指示文）をすべて公开します。
以下のプロンプトを ChatGPT や Claude にコピー&ペースするだけで使えます。

### 🔥 プロンプト①：SPI攻略プラン作成

```
あなたはSPI対策のプロフェッショナルです。
私にspiの模拟試験を実施し、出題傾向を分析してください。
その上で、私に最適な学習プランを作成してください。
現在のレベル: 初級（偏差値50前半）
学习時間: 每日1時間
目标: 偏差値65达成
```

このプロンプトで自分に 맞춘学習计划が手に入ります。

### 🔥 プロンプト②：頻出問題の生成

```
SPIの非言語（数的処理）で頻出のテーマを教えてください。
각テーマごとに3問ずつ、难的度は中级的問題を生成してください。
解答と解说もお願いします。
テーマ: 損益算、速度算、場合の数
```

これ一冊分の練習問題が、AIで無限に生成できます。

### 🔥 プロンプト③：弱点克服スクリプト

```
私の出した答えを检证してください。
もし間違っていれば、错的のポイントを很简单に解释し、
类似の応用問題も1問出してください。
正答率が上がるまで繰り返し问题を出題してください。
```

错误問題を放置せず、都度解决できます。

### 🔥 プロンプト④：模擬試験の作成

```
実際のSPI試験と同様の形式で、30問の模擬問題を作成してください。
制限時間は35分です。
言語・非言語各大15問ずつ占比で作成してください。
解答后将、あなたの偏差値も估算してください。
```

自分の реаль的水平 度に合っている模擬問題が作れます。
